# Svelte setup

Svelte init setup.

## Tech Stack

- Svelte
- Rsbuild
- Tailwind CSS
- TypeScript
- ESLint & Prettier
- pnpm

## Project Layout (important files)

- `src/` — application source (e.g. `App.svelte`, `index.ts`)
- `rsbuild.config.ts` — build configuration
- `postcss.config.mjs` — PostCSS config (Tailwind integration)
- `index.css` — global styles
